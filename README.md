# FSDH - Full Stack Dermatology Hospital Website

A modern, responsive healthcare web application built with React, designed to deliver a seamless digital experience for patients exploring hospital services, doctors, and medical facilities.

## Overview

FSDH is a full-stack dermatology hospital website that provides a clean, structured, and user-friendly interface for patients to explore medical services, book appointments, and connect with healthcare providers.

### Features

- Responsive design across mobile, tablet, and desktop
- Modern UI/UX with smooth animations
- Department and service listings
- Doctor profiles and specialties
- Online appointment booking system
- Emergency contact and location information
- Google Maps integration

## Tech Stack

- **Frontend:** React.js, JavaScript, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide Icons
- **Build Tool:** Vite

## Getting Started

```bash
npm install
npm run dev
```
